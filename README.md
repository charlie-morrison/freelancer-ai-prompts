# Freelancer AI Prompts

A collection of practical AI prompts for freelancers — covering client acquisition, pricing, project management, and professional communication. Copy-paste ready with clear placeholders.

Whether you're just starting out or scaling your freelance business, these prompts turn AI into your business partner.

## Client Acquisition

### Ideal Client Profile
```
I'm a freelance [your specialty]. Help me define my ideal client profile:
- Industry/niche they're in
- Company size and budget range
- Pain points I can solve
- Where they hang out online
- How they typically find freelancers

My skills: [list skills]
My experience level: [junior/mid/senior]
My preferred project types: [list]
```

### Cold Outreach Email
```
Write a cold outreach email to a potential client. Context:
- I'm a freelance [specialty]
- The company is [company name/description]
- I noticed they [specific observation about their business]
- I can help them with [specific value proposition]

Rules: Keep under 150 words. No generic flattery. Lead with value.
Include one specific suggestion they can act on immediately.
```

### Upwork Proposal Generator
```
Write an Upwork proposal for this job posting:
[paste job description]

My relevant experience: [your experience]
My approach: [how you'd tackle it]

Rules: First line must hook them. Show you read the posting
(reference specific requirements). Include one smart question
that shows expertise. Keep under 200 words.
```

### Portfolio Case Study Writer
```
Help me write a portfolio case study for a project:
- Client industry: [industry]
- Problem: [what they needed]
- My solution: [what I built/delivered]
- Results: [measurable outcomes]

Format: Challenge → Approach → Solution → Results.
Keep it scannable with bullet points. Under 300 words.
```

## Pricing & Negotiation

### Rate Calculator
```
Help me calculate my freelance rate:
- Target annual income: $[amount]
- Desired work hours per week: [hours]
- Weeks off per year: [weeks]
- Business expenses (monthly): $[amount]
- Tax rate: [percentage]
- Utilization rate (billable vs non-billable): [percentage, typically 60-75%]

Calculate my minimum hourly rate, recommended hourly rate
(with profit margin), and equivalent project-day rate.
```

### Value-Based Pricing
```
Help me price this project based on value, not hours:
- Client's business: [description]
- What they need: [project scope]
- Expected business impact: [revenue increase, cost savings, etc.]
- My estimated effort: [hours]
- My hourly rate: $[rate]

Calculate a value-based price that's fair for both sides.
Explain the ROI to the client.
```

### Counter-Offer Response
```
A client offered $[amount] for a project I quoted at $[amount].
Help me respond:
- I want to close this deal but protect my rate
- Options: negotiate scope (not rate), offer payment terms,
  add value instead of discounting
- Maintain a collaborative (not adversarial) tone
- Include a firm but friendly bottom line

Write the actual response I can send.
```

## Project Management

### Project Scope Definer
```
Help me define the scope for a freelance project:
- Client wants: [their request]
- My understanding of the deliverables: [your interpretation]
- Timeline they mentioned: [timeline]
- Budget range: [if known]

Create a clear scope document with:
1. In-scope items (specific deliverables)
2. Out-of-scope items (explicitly excluded)
3. Assumptions
4. Dependencies (what I need from the client)
5. Milestones and delivery dates
```

### Risk Assessment
```
I'm starting a freelance project: [description]
Client type: [new/returning, company size]
Timeline: [deadline]
Budget: [amount]

Identify top 5 risks and for each:
- Likelihood (high/medium/low)
- Impact (high/medium/low)
- Mitigation strategy
- Contingency plan
```

### Scope Change Request Handler
```
My client wants to add [new feature/change] to the project.
Original scope: [brief description]
Current progress: [percentage/phase]
Original budget: $[amount]

Help me:
1. Assess the impact (time, cost, quality)
2. Write a professional change request response
3. Offer options (absorb, re-scope, additional budget)
```

## Client Communication

### Weekly Status Update
```
Write a professional weekly status update email for my client:
- This week I completed: [tasks]
- Currently working on: [tasks]
- Blockers: [if any]
- Next week's plan: [tasks]
- Any decisions needed from client: [items]

Keep it scannable. Use bullet points. Under 150 words.
```

### Red Flag Detector
```
A potential client told me: "[paste their message/requirements]"

Analyze this for red flags:
- Scope creep risk
- Budget concerns
- Communication style issues
- Unrealistic expectations
- Missing information I should ask about

Rate the lead: green (proceed), yellow (proceed with caution),
red (walk away).
```

### Late Payment Follow-Up
```
Write a follow-up message for a late payment:
- Invoice amount: $[amount]
- Due date: [date]
- Days overdue: [X]
- Previous reminders sent: [number]
- Relationship importance: [high/medium/low]

Escalation level: [friendly reminder / firm follow-up / final notice]
Include specific next steps and consequences if applicable.
```
